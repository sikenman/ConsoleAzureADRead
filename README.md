# ConsoleAzureADRead
## Purpose
Get list of Group for give User in Azure AD (Microsoft Entra ID)

*Libraries*
* Azure.Identity (1.10.4)
* Microsoft.Graph (5.46.0)
* Newtonsoft.Json (13.0.3)
